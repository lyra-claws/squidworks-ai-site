# squidworks.ai (v1)

Static site + custom intake form + Cloudflare Pages Function gateway.

## Goals
- Premium-looking marketing site (fast, simple)
- Custom intake form UI
- Store-first intake submissions to Cloudflare D1
- Sync to GoHighLevel (GHL) via API

## Structure
- `public/` static site files
- `functions/` Cloudflare Pages Functions
